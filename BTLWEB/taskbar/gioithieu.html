GENERAL INTRODUCTION

1. Reasons for Choosing the Topic

Nowadays, data appears in every field: education, business, finance, healthcare, and more. However, if data is not presented in a visual and intuitive way, it becomes difficult for users to interpret and make decisions. Building a data visualization website helps users better understand information, thereby improving learning and work efficiency.

2. Objectives

  • Develop a simple and effective data visualization website.
  • Support easy data input (manually or via file upload).
  • Allow users to choose chart types and customize colors, titles, etc.
  • Provide a user-friendly and easy-to-use interface.

3. Target Users and Technologies Used

  • Target users: Students, office workers, general users.
  • Technologies used: HTML, CSS, JavaScript, Canva, etc.
